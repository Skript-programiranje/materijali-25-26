# Vežbe iz Skript programiranja

Zvanični materijali za pripremu praktičnog ispita kursa Skript programiranje u akademskoj 2025/26. godini.
Materijali su napravljeni od kodova sa održanih vežbi.

Više informacija se može naći na [zvaničnoj stranici kursa][sp-webpage].

Časovi su podeljeni u direktorijume, a detaljnije o gradivu časa se može naći u odgovarajućim `notes.md` fajlovima.

[sp-webpage]: https://www.skript-programiranje-i.matf.bg.ac.rs/
